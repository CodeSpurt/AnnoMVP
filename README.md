This app is a demonstration on how to use 'MVP' in Android.<br />

<b>Library used:</b> AnnoMVP-1.0<br />

<b>build.gradle</b><br />
project: classpath 'com.neenbedankt.gradle.plugins:android-apt:1.8'<br />
app:
dependencies {
    apply plugin: 'com.neenbedankt.android-apt'
}
